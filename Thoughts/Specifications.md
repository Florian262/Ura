The current dark, low-contrast grid creates a "passive registry" mental block—it looks like a database admin dashboard, which signals to the brain that it is a data-tracking tool rather than an active learning space.Here is how pedagogical science can reshape this layout to trigger an active learning response, followed by a human-designed "Studio Print" UI blueprint.1. What Cognitive Science Says About Active Learning LayoutsA. The Saccadic Eye Movement & Signaling Principle (Mayer, 2014)The Issue: The 6-box uniform grid forces the user's eyes to jump erratically across identical geometric zones. There is no clear visual anchor point, causing high cognitive friction.The Fix: A Linear Progressive Roadmap with Spatial Hierarchy. Instead of a uniform grid, the interface should map out a clear vertical or progressive pathway. The active level should dominate the screen visually (Large Spatial Anchoring), while future levels are visually sub-staged, reducing immediate visual clutter and showing a clear "mountain to climb."B. Cognitive Load Theory: Dark vs. Light for Text Density (Sweller)The Issue: High-density, fine-print technical data is highly exhausting to read on pure black backgrounds over long periods because it forces the pupils to dilate wider, causing blurred word edges (halation).The Fix: An Editorial Light Mode Canvas (#FBFBF9 warm paper bone / cream). Science shows that high-contrast, dark-on-light interfaces mimic physical learning media (books, research papers), priming the brain for critical reading, analytical processing, and deep focus.C. Generative Learning Theory: Scaffolding Motivation (Fiorella & Mayer)The Issue: Your current layout shows raw numbers: [Mësimet: 2] 0%. This provides statistical tracking, not learning motivation.The Fix: Shift from raw system metrics to Goal-Oriented Preview Teasers. Show a small, explicit curiosity gap for the level (e.g., instead of just "FILL ESTAR", add a tiny tagline: "Mëso të prezantosh veten dhe të përdorësh mbi 200 fjalë të përbashkëta Ballkanike"). This activates pre-existing knowledge and primes goal-oriented motivation.2.
